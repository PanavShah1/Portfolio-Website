Track Astroids

This program requires the user to type in a date and a timezone. Based on the date and timezone it will call an api from NASA. It will show and plot all the astroids that were at their point of closest approach on that day.

My friend and I made this website for the Cyber Cypher hackathon. The problem statement was to develop a solution to enhance patient experience and efficiently book appointments with doctors.

Stock predictor

Our team submitted this application for the IIT Bombay's HackAI hackathon during the 2023 techfest. It is an application which uses the top stocks and fetches historical data of them. It uses that to predict a price for the future using linear regression.
